# Como rodar ?
**Na raiz do projeto**  
crie o arquivo .env  
Adicione as variaveis abaixo
```
DB_CON="url conexao mongoDB"  
PORT=numero porta
```

```bash
$ yarn install (aguarde)

$ yarn rundev (para rodar o server)

```
